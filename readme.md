# My Site

これは HTML + CSS + JavaScript で作る静的サイトのサンプルです。

## 📁 構成

- `index.html` … メインページ
- `header.html` / `footer.html` … 共通パーツ
- `style.css` … 共通スタイル
- `include.js` … 動的読み込みスクリプト

## 🚀 使用方法

1. `Live Server` などで HTTP 起動
2. ブラウザで `index.html` を開く
